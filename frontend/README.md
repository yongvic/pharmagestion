# PharmaGestion — Frontend

Interface React de PharmaGestion. Voir le [README principal](../README.md) pour l'installation et le lancement.

## Scripts locaux

```bash
npm run dev      # Vite dev server (port 5173)
npm run build    # Build production → dist/
npm run lint     # ESLint
```

En développement, préférez `npm run dev` depuis la **racine du projet** (lance aussi le backend et Electron).
